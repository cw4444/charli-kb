# Overview

`charli-kb` is a plain Markdown knowledge base that turns curated raw material into a small set of durable, linked notes.

The first test ingest covered a batch of Dan Koe material on focus, creativity, identity change, and project-based self-direction. The useful synthesis is that a knowledge system should not reward hoarding. It should reward live projects, clear questions, and ideas that continue to pay rent.

Current working rule: Notion remains the messy capture dump; this wiki only keeps material that supports a project, question, or durable synthesis.
